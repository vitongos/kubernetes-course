# Demos y fuentes para el curso de Docker/Kubernetes

## Prerrequisitos

Para ejecutar las demos es necesario tener instalados:

- Git
- Docker
  - Docker Machine
  - Docker Compose
- VirtualBox
- Kubernetes
  - kubectl
  - Minikube
  - kubeadm
